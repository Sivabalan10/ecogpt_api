EcoGPT API - Save Energy for Future

API Documentation

This document outlines the usage of the API endpoints for the project.

1. User Login

Endpoint: /user_login

Method: POST

Request Body:

{
    "username": "user123",
    "password": "mypassword"
}

Response:

{
    "status": "success",
    "message": "Login successful",
    "user_data": { ... }
}

2. User Logout

Endpoint: /user_log_out

Method: POST

Response:

{ "status": "success", "message": "User logged out successfully" }

3. Get User Data

Endpoint: /get_all_user_table_data/<user_id>

Method: GET

Response:

{ "status": "success", "user_data": { ... } }

4. Check-in

Endpoint: /check_in

Method: POST

Request Body:

{
    "user_id": 1,
    "lat": 12.345,
    "lon": 67.890
}

Response:

{ "status": "success", "message": "Check-in successful" }

5. Check-out

Endpoint: /check_out

Method: POST

Request Body:

{
    "user_id": 1,
    "lat": 12.345,
    "lon": 67.890
}

Response:

{ "status": "success", "message": "Check-out successful" }

6. View Reports

Endpoint: /view_reports/<user_id>

Method: GET

Response:

{ "status": "success", "records": [...] }

7. Make Transaction

Endpoint: /make_transaction

Method: POST

Request Body:

{
    "user_id": 1,
    "product_name": "Product A",
    "cc_points": 50
}

Response:

{ "status": "success", "message": "Transaction successful" }

8. Power Unit Saved

Endpoint: /power_unit_saved

Method: GET

Response:

{ "status": "success", "total_power_saved": 100 }
